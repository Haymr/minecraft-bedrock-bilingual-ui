# How to Use the Bilingual UI Addon

Welcome! This addon allows you to view Minecraft Bedrock Edition in two languages at the same time.

## Installation Steps
1. Download the `Bilingual_UI.mcpack` file.
2. If you are on an iOS device (iPhone/iPad) or Android, simply tap the file and select "Minecraft" from the share menu.
3. If you are on Windows 10/11, double-click the file to import it into Minecraft.
4. Once the game says "Import Successful", go to **Settings > Global Resources**.
5. Find "Bilingual Bedrock UI" under "My Packs" and click **Activate**.

## How It Works
- The primary language (e.g., English) will appear on top as normal.
- The secondary language (e.g., French) will appear directly below it in a grayish, italicized font.
- This addon uses native game features, meaning it won't break your achievements or require experimental gameplay!

Enjoy playing Minecraft with dual languages!
