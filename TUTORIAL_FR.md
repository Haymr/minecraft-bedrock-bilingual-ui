# Interface Bilingue pour Minecraft Bedrock - Guide d'installation iOS

Ce mod fonctionne sans dépendance au Marketplace. Veuillez suivre ces instructions spécifiques pour une installation propre sur iOS:

1. **Téléchargement:** Téléchargez le fichier avec l'extension `.mcpack` via Safari.
2. **Préparation Critique:** Fermez complètement l'application Minecraft. Si l'application tourne en arrière-plan, l'importation risque de ne pas se lancer correctement.
3. **Importation:** Ouvrez l'application "Fichiers" (Files) sur votre appareil iOS. Touchez le fichier `.mcpack` une seule fois.
4. Si Minecraft ne s'ouvre pas automatiquement, maintenez votre doigt sur le fichier, choisissez "Partager", puis sélectionnez Minecraft dans la liste.
5. **Activation:** Une fois dans le jeu, un message "Importation réussie" apparaîtra. Allez dans Paramètres > Ressources globales (Global Resources) et activez le pack. Le jeu rechargera l'interface et vos textes bilingues seront affichés avec leur formatage italique et indenté spécifique.
