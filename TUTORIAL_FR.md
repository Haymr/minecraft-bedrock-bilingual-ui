# Comment utiliser l'Addon Interface Bilingue

Bienvenue ! Cet addon vous permet de visualiser Minecraft Bedrock Edition dans deux langues simultanément.

## Étapes d'installation
1. Téléchargez le fichier `Bilingual_UI.mcpack`.
2. Si vous utilisez un appareil iOS (iPhone/iPad) ou Android, appuyez simplement sur le fichier et sélectionnez "Minecraft" dans le menu de partage.
3. Si vous êtes sur Windows 10/11, double-cliquez sur le fichier pour l'importer dans Minecraft.
4. Une fois que le jeu affiche "Importation réussie", allez dans **Paramètres > Ressources globales**.
5. Trouvez "Bilingual Bedrock UI" sous "Mes packs" et cliquez sur **Activer**.

## Comment ça marche ?
- La langue principale (par exemple l'anglais) apparaîtra en haut normalement.
- La langue secondaire (par exemple le français) apparaîtra directement en dessous, dans une police grise en italique.
- Cet addon utilise des fonctionnalités natives du jeu, ce qui signifie qu'il ne bloquera pas vos succès (achievements) et ne nécessitera pas d'activer le mode de jeu expérimental !
